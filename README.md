# IDABluez

To install place idabluez/theme.css to IDADIR/themes/idabluez, idabluez/user.css to USERDIR/AppData/Roaming/Hex-Rays/IDA Pro/themes/idabluez and then select idabluez theme in IDA Options->Colors
